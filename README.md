# spesafamiglia
🛒 Spesa Famiglia è un'app web condivisa in tempo reale per gestire la lista della spesa di tutta la famiglia. Sviluppata da zero in meno di un mese, offre login sicuro, categorie intelligenti, foto sugli articoli, calcolo automatico dei costi e aggiornamenti mensili con novità sempre nuove. Un progetto semplice ma ambizioso! 💪🎉
